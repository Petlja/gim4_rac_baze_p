Агрегатне функције и груписање - квиз¶

    Q-67: Којом агрегатном функцијом добијамо просечну вредност неке колоне?

  • AVG
  • AVERAGE
  • MEAN
  • MEDIAN

Дат је следећи упит:

SELECT id_ucenik, count(*) AS br_izost
FROM izostanak
GROUP BY id_ucenik
ORDER BY br_izost DESC
LIMIT 3;

    Q-68: Шта се добија извршавањем овог упита?

  • Подаци о прва три ученика са различитим бројем изостанака.
  • Подаци о ученицима који су направили прва три изостанка.
  • Подаци о три ученика са највише изостанака.
  • Подаци о прва три ученика који имају изостанке.
    Q-69: Упари упите са њиховим описима
  • SELECT COUNT(*)
    FROM izostanak
  • Укупан број направљених изостанака
  • SELECT COUNT(*)
    FROM izostanak
    GROUP BY id_ucenik
  • Број изостанака сваког ученика
  • SELECT COUNT(DISTINCT id_ucenik)
    FROM izostanak
  • Број ученика који су правили изостанке

Дати су следећи упити:

/* upit broj 1*/
SELECT id_ucenik, id_predmet, max(ocena) as max_ocena
FROM ocena
WHERE max_ocena < 3
GROUP BY id_ucenik, id_predmet

/* upit broj 2*/
SELECT id_ucenik, id_predmet, max(ocena) as max_ocena
FROM ocena
GROUP BY id_ucenik, id_predmet
HAVING max_ocena < 3

/* upit broj 3*/
SELECT id_ucenik, id_predmet, max(ocena) as max_ocena
FROM ocena
HAVING max_ocena < 3
GROUP BY id_ucenik, id_predmet
