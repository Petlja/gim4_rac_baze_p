    Дат је следећи упит:
    
    .. code-block:: sql
    
        SELECT naziv, CASE
                WHEN razred = 1 THEN 'I'
                WHEN razred = 2 THEN 'II'
                WHEN razred = 3 THEN 'III'
                WHEN razred = 4 THEN 'IV'
            END AS godina
        FROM predmet;
