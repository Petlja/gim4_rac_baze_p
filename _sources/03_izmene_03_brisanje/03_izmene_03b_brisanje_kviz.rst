        :answer_b: ERASE
        :answer_c: DELETE
        :answer_d: DROP
        :correct: c

        Којом SQL командом се бришу одабрани подаци из табеле?

.. quizq::

    .. mchoice:: brisanje_sta
        :answer_a: Изабрани редови табеле.
        :answer_b: Изабране колоне табеле.
        :answer_c: Изабране колоне изабраних редова табеле.
        :answer_d: Цела табела се уклања из структуре базе.
        :correct: a

        Шта се од података брише SQL командом за брисање података из табеле?

.. quizq::

    .. mchoice:: brisanje_greska
        :multiple_answers:
        :answer_a: Поништавањем извршења последњег упита помоћу команде UNDO.
        :answer_b: Тако што пре упита брисања извршимо упит SELECT, да бисмо видели редове који би били обрисани.
        :answer_c: Тако што пре упита брисања направимо копију целе базе података.
        :answer_d: Тако што погледамо привју резултата упита брисања.
        :correct: b,c
